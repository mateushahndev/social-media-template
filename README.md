# Social Media Template

Interface estática inspirada em layout de rede social, desenvolvida com foco em responsividade e organização estrutural utilizando HTML e CSS puros.

## Contexto

Este projeto foi desenvolvido como prática de layout responsivo a partir de uma referência visual.  
O objetivo principal foi reproduzir a estrutura da interface manualmente, sem copiar código, consolidando fundamentos de CSS e organização de componentes visuais.

## Objetivos Técnicos

- Estruturação semântica com HTML5
- Construção de layout utilizando Flexbox
- Criação de múltiplas seções (menu lateral, feed, eventos, anúncios, contatos)
- Implementação de barra de navegação fixa
- Organização de interface com múltiplos blocos reutilizáveis
- Aplicação de media queries para adaptação em diferentes resoluções

## Responsividade

O layout foi ajustado para diferentes breakpoints, reorganizando:

- Menu lateral
- Feed principal
- Barra de navegação
- Seção de anúncios e contatos

A interface se adapta progressivamente até dispositivos móveis.

## Tecnologias Utilizadas

- HTML5
- CSS3
- Flexbox
- Media Queries
- Font Awesome (CDN)

## Observação

Projeto desenvolvido como exercício prático de consolidação de fundamentos em HTML e CSS.  
Não possui integração com backend ou funcionalidades dinâmicas.
